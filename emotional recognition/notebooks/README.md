# Jupyter Notebook 示例

本目录包含 Jupyter Notebook 示例，用于演示如何使用本项目。

## 建议的 Notebook 列表

1. **01_data_exploration.ipynb** - 数据探索和分析
2. **02_model_training.ipynb** - 模型训练示例
3. **03_model_evaluation.ipynb** - 模型评估和可视化
4. **04_prediction.ipynb** - 使用训练好的模型进行预测

## 启动 Jupyter Notebook

```bash
jupyter notebook
```

或
